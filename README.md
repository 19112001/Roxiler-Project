# How to start project

## 1. git clone https://github.com/19112001/Roxiler-Project.git

## 2. cd Roxiler-project

## 3. cd backend/

## 4. npm install

## 5. create a mongoDB database "product_transactions"

## 6. npm run seed

## 7. npm run start

## 8. open new terminal and cd frontend/

## 9. npm install

## 10. npm start
